# 流浪貓冒險｜遊戲美術 Prompt 風格規範

> 目的：建立整個遊戲素材的共用 prompt 規範，讓背景、角色、UI、道具、特效在同一套美術語言下產出。  
> 遊戲畫面：直式 16:9。素材比例可依用途使用 1:1、16:9、4:3。角色與可互動素材預設透明背景。

---

## 1. 核心風格一句話

美式簡約插畫風格的 2D 遊戲美術，明亮高彩度、乾淨大色塊、輕微手繪筆觸、粉紅與水藍街頭色系，童趣、滑稽、冒險感，但不要日系、不要寫實、不要過度細節。

---

## 2. 固定風格 Prompt

所有素材 prompt 前段建議固定使用：

```md
美式簡約插畫風格的 2D 遊戲素材，明亮高彩度，乾淨大色塊，輕微手繪筆觸，粉紅、桃紅、水藍、青綠、藍紫色街頭配色，圓潤可愛但不幼稚，滑稽、有冒險感，藍紫色塊陰影，簡潔清楚的輪廓，適合 Godot / H5 直式遊戲使用。
```


## 3. 色彩規範

### 主要色彩
- 天空：亮青藍、飽和水藍。
- 建築：桃紅、粉紅、紫粉、青綠、藍綠。
- 陰影：藍紫、深水藍，不使用寫實黑影。
- 點綴：奶油白雲朵、少量黃橘光源、綠色植物。

### 避免
- 髒灰、低飽和、黑暗厚重、賽博龐克、寫實金屬、過度漸層。
- 不要日系萌系、不要動漫線稿、不要厚重黑邊。

---

## 4. 造型與筆觸規範

- 造型以大塊面、簡化幾何、圓潤比例為主。
- 細節只保留能辨識角色與功能的部分。
- 筆觸可有 10–20% 手繪感，不要油畫感、不要照片感。
- 角色表情要誇張清楚，適合小尺寸辨識。
- 敵人可以可惡、滑稽、怪誕，但不要恐怖血腥。
- UI 要厚實、圓角、像玩具或街頭招牌，不要科技感介面。

# 7. 輸出規格

## 遊戲畫面
- 主畫面為直式 16:9。
- 背景圖建議優先輸出直式 16:9。
- 可依素材用途使用 1:1、4:3、16:9。

## Sprite
- 預設透明背景。
- 單一角色完整身體，不裁切。
- 保留 8–12% 安全邊距，避免動畫裁切。
- 同一角色的不同動作要固定：頭身比例、配色、服裝、眼睛形狀、主要特徵。
- 需要明確標註方向：面向左 / 面向右 / 正面。

## UI
- UI 元件預設透明背景。
- 不直接生成文字，文字交由遊戲或設計工具排版。
- 按鈕需保留足夠中文字空間。
- 同一組 UI 的 default / pressed / disabled 狀態，尺寸與造型要一致。

## 背景
- 背景避免角色與文字。
- 中央需保留遊戲可讀區，不放過多高對比雜物。
- 遠景細節降低，前景可有垃圾桶、圍籬、水漬、紙箱、植物等街頭物件。
- 不要讓背景細節搶過角色與 UI。

# 9. 組合公式

每次產圖建議照這個順序寫：

```md
固定風格描述 + 素材類型 + 主體內容 + 外觀特徵 + 動作 / 狀態 + 構圖需求 + 輸出規格 + 負面 prompt
```

範例：

```md
美式簡約插畫風格的 2D 遊戲角色 sprite，明亮高彩度，乾淨大色塊，輕微手繪筆觸，粉紅、水藍、青綠、藍紫街頭配色，圓潤可愛但不幼稚。一隻流浪貓，表情機靈勇敢，正在向右奔跑，尾巴高高翹起，動作有速度感，完整身體，不裁切，無背景，透明背景，無文字，比例 1:1。避免：photorealistic, 3D render, anime, heavy black outline, overly detailed, text, watermark。
```
