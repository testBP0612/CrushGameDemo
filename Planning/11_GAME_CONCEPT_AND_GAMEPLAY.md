# 《Meow準快跑》遊戲與玩法說明文件

---

## 一、我們做的是什麼遊戲？遊戲理念是什麼？

《Meow準快跑》是一款以流浪貓視角出發的手機直式 H5 遊戲，核心玩法採用「風險撤離型」設計，也就是玩家每一回合都要判斷：現在安全撤退，還是繼續挑戰下一關、追求更高收益。

遊戲世界觀設定在明亮、高彩度的街頭場景中。玩家扮演一隻在城市中流浪的貓咪，為了搶奪食物與金幣，需要面對各種「貓咪眼中的街頭怪物」。這些怪物並不是傳統奇幻敵人，而是將城市日常與在地諧音梗轉化成幽默角色，例如流浪小黃狗、小哥鴿、大賽鴿、哥布林阿嬤、蜥yee、安鼠之亂、當作我塑膠、咪咪冒冒、社畜哥布林等。

**整體理念：** 將 Crash Game / Push-your-luck 的刺激決策，重新包裝成可愛、幽默、低門檻的流浪貓街頭冒險。

我們保留了「高風險、高回報、短回合、強決策」的核心張力，但透過明亮可愛的視覺、貓咪主角、在地化怪物與街頭奇遇事件，降低傳統博弈玩法的距離感，讓遊戲更像一款休閒街機冒險，而不是冷冰冰的倍率遊戲。

---

## 二、玩家核心玩法

玩家進入遊戲後，可以投入虛擬金幣開始挑戰。每次挑戰成功擊敗怪物後，關卡進度、倍率與目前收益都會提升。此時玩家會面臨兩個選擇：

1. **撤退領取** — 立即帶走目前收益，安全結束本局。
2. **挑戰下一隻** — 繼續冒險，追求更高倍率與更高收益，但若挑戰失敗，本局收益可能歸零或損失投注金額。

遊戲流程：

```
下注 / 投入金幣
↓
開始挑戰
↓
擊敗怪物
↓
收益與倍率提升
↓
選擇撤退或挑戰下一隻
↓
撤退成功 / 挑戰失敗
↓
結算並查看排名
```

除了基本挑戰流程，遊戲也加入了輕量隨機事件，例如「虎爺降臨」。當事件觸發時，虎爺會突然登場，直接幫玩家擊飛怪物，並帶來獎金加碼。這類事件不會讓遊戲變得複雜，但能讓每一局多一點驚喜與故事性。

此外，遊戲也設計了排行榜式非同步對戰。玩家不需要即時連線，也不需要進入房間，只要完成一局，就能透過排行榜看到自己的收益、最深關卡、排名與超越多少玩家，形成低成本但有效的競爭感。

## 三、這款遊戲的核心樂趣是什麼？

《Meow準快跑》的核心樂趣可以拆成四個層次。

### 一、何時收手的心理張力

遊戲最核心的樂趣是「要不要再拚一次」。

玩家每打贏一關，都會看到目前收益增加，但同時也知道繼續挑戰可能會失敗。這種「已經賺到了，但好像還可以更多」的心理拉扯，是遊戲最主要的刺激來源。

```
現在撤退是不是太早？
再打一隻會不會更賺？
如果下一關失敗，我會不會後悔？
```

### 二、可愛包裝降低門檻

傳統 Crash Game 容易給人博弈感較重、視覺壓力較高、玩法較冰冷的印象。《Meow準快跑》用貓咪、街頭、食物、怪物諧音梗與高彩度 UI 重新包裝後，讓同樣的風險決策變得更親民、更有趣。

玩家不是在看一條抽象倍率曲線，而是在幫一隻流浪貓判斷：

```
要不要繼續搶食物？
要不要再打一隻街頭怪物？
要不要現在帶著戰利品逃跑？
```

### 三、怪物與事件帶來新鮮感

遊戲不是只有倍率變化，而是透過怪物設計與街頭奇遇事件提高耐玩度。

每一隻怪物都來自貓咪視角的城市想像，像是狗、鴿子、老鼠、塑膠袋、阿嬤、社畜人類，都被轉化成幽默敵人。玩家會自然產生好奇心，想知道下一關會遇到什麼怪物。

輕量隨機事件則讓每一局都有可能發生突發轉折。例如虎爺降臨、獎金加碼、怪物被直接擊飛等，都能讓玩家產生「這局很特別」的記憶點。

### 四、排行榜讓單人流程產生競爭感

透過排行榜式非同步對戰，遊戲把單局結果轉化成可比較的成績。

玩家撤退成功時，可以看到目前排名、超越多少玩家、個人最高紀錄；玩家挑戰失敗時，也能看到本次最深進度與排名表現。這讓失敗不只是歸零，而是仍然保留「我衝到第幾關」的成就感。

```
我能不能超過朋友？
我能不能刷新自己的最高紀錄？
我再玩一局會不會進排行榜？
```

## 四、未來可商業化或擴充方向

### 一、排行榜與社交競爭

可以串接 Google 登入或 Firebase，建立正式排行榜，讓玩家可以保存個人紀錄，並與朋友、同事或全服玩家競爭。

未來可擴充：
- 今日排行榜 / 本週排行榜 / 好友排行榜
- 公司內部挑戰榜
- 最高收益榜 / 最深關卡榜
- 最敢衝玩家稱號

### 二、每日挑戰與同種子挑戰

未來可以加入每日固定挑戰。所有玩家面對同一組怪物與風險序列，比較誰能在相同條件下做出最佳撤退判斷。

```
今天你打到第幾關？
你在哪一關撤退？
你是不是貪心爆掉了？
```

### 三、更多街頭怪物與在地化內容

未來可持續擴充以貓咪視角出發的街頭怪物，例如：
- 夜市系怪物
- 巷口阿伯系怪物
- 機車騎士系怪物
- 外送員系怪物
- 校園貓敵人
- 捷運站街頭事件

### 四、街頭奇遇事件擴充

未來可擴充更多事件類型：
- **保命型**：失敗時保留部分收益
- **加碼型**：下一關成功獎勵提高
- **情報型**：提示下一隻怪物危險度
- **趣味型**：立即獲得小額獎勵或特殊演出

### 五、品牌合作與活動化

遊戲以貓咪、街頭、食物為核心，適合進行活動合作：
- 寵物品牌聯名 / 貓咪公益活動
- 罐罐 / 飼料品牌合作
- 公司內部活動競賽
- 節慶限定怪物與事件
- H5 行銷活動頁

---

## 五、總結

《Meow準快跑》是一款將風險撤離玩法可愛化、在地化、故事化的 H5 遊戲。它保留了 Crash Game 類型最具商業價值的短回合、高張力與「再玩一局」特性，但透過流浪貓世界觀、幽默怪物、街頭奇遇與排行榜競爭，降低傳統博弈感，提升親和力與重玩動機。

> **本作的核心不是「賭」，而是「要不要再冒一次險」。**
> 玩家每一次選擇撤退或繼續，都是一次簡單但刺激的判斷；而可愛角色、特殊事件與排行榜，則讓這個判斷變得更有故事、更有記憶點，也更有擴充潛力。
